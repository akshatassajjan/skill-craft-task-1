Responsive Landing Page
SkillCraft Technology Internship - Task 01

A modern and responsive landing page developed as part of the SkillCraft Technology Web Development Internship.

📌 Project Description

This project demonstrates a responsive landing page with an interactive navigation menu. The navigation menu changes its appearance when the user scrolls through the page and when hovering over menu items.

The navigation bar remains fixed at the top of the page, ensuring easy access to all sections. The website is fully responsive and provides an optimal viewing experience across desktops, tablets, and mobile devices.

🚀 Features
Fixed Navigation Bar
Scroll-Based Navbar Color Change
Hover Effects on Navigation Links
Smooth Scrolling Navigation
Responsive Design
Modern User Interface
Mobile-Friendly Layout
Interactive Animations
Clean and Well-Structured Code

🛠️Technologies Used
HTML5
CSS3
JavaScript (ES6)
