Responsive Landing Page<br>
SkillCraft Technology Internship - Task 01<br>

A modern and responsive landing page developed as part of the SkillCraft Technology Web Development Internship.
<br>
📌 Project Description<br>

This project demonstrates a responsive landing page with an interactive navigation menu. The navigation menu changes its appearance when the user scrolls through the page and when hovering over menu items.<br>

The navigation bar remains fixed at the top of the page, ensuring easy access to all sections. The website is fully responsive and provides an optimal viewing experience across desktops, tablets, and mobile devices.<br>

🚀 Features<br>
Fixed Navigation Bar<br>
Scroll-Based Navbar Color Change<br>
Hover Effects on Navigation Links<br>
Smooth Scrolling Navigation<br>
Responsive Design<br>
Modern User Interface<br>
Mobile-Friendly Layout<br>
Interactive Animations<br>
Clean and Well-Structured Code<br>
